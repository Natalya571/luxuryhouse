## 🚀 Как использовать

### Вариант 1: GitHub Pages (рекомендуется)
1. Опубликуйте архив в GitHub-репозиторий
2. Перейдите: **Settings → Pages**
3. Source: **Deploy from a branch**
4. Branch: **main** → **/ (root)**
5. Нажмите **Save**
6. Через 1-2 минуты сайт будет доступен по адресу `https://{ваш-логин}.github.io/{репозиторий}/`

### Вариант 2: Локальный просмотр
1. Распакуйте ZIP-архив
2. Откройте `index.html` в браузере
3. Все страницы и изображения загружаются локально — интернет не нужен

### Вариант 3: На собственном хостинге
1. Загрузите все файлы на любой хостинг или FTP
2. Сайт работает как статический — PHP не требуется
